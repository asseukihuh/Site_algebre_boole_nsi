I. Internet, web et langage HTML

  1. Qu'est-ce qu'internet ? Qu'est-ce que le web ?

    2. Internet est un réseau d'ordinateurs dans le monde connectés entre eux,
    cela permet la transmission d'informations et de données.
    Il permet également à des utilisateurs de se connecter et de communiquer
    entre eux à travers le monde.

  2. Citer quelques applications d'internet.

    3. Il existe plusieurs applications d'internet comme les e-mails,
    la recherche en ligne, les jeux en ligne,
    les réseaux sociaux etc...

  3. Qu'est-ce que le langage HTML ? Comment fonctionne-t-il ?

    4. Le HTML ou en français le balisage hypertexte est un langage de
    balisage utilisé pour créer des pages Web.
    Il permet de structurer le contenu en utilisant des balises qui définissent les différents éléments.
    Les balises entourent le contenu et permettent au navigateur Web de comprendre comment
    afficher la page. Le HTML est souvent utilisé avec d'autres langages tels que le CSS et le JavaScript
    pour ajouter des styles et des fonctionnalités interactives.

  4. Lister à l'aide des balises <ul> et <li> les balises HTML utilisées dans ce document en expliquant à quoi elles servent.
    • La balise html indique le début et la fin d'un document HTML.
    • La balise head contient des informations sur la page Web,
      telles que le titre de la page, les méta-données et les liens vers d'autres fichiers.
    • La balise meta fournit des métadonnées supplémentaires sur la page,
      telles que la description, les mots-clés et l'encodage.
    • La balise link est utilisée pour relier la page à d'autres fichiers,
      tels que des feuilles de style CSS ou des icônes de site Web.
    • La balise script est utilisée pour inclure du code
      JavaScript dans une page Web.
    • La balise title définit le titre de la page Web,
      qui est affiché dans l'onglet du navigateur.
    • La balise body contient le contenu principal de la page Web,
      tel que le texte, les images et les liens.
    • La balise div est utilisée pour diviser une page Web en
      sections distinctes et appliquer des styles à ces sections.
    • La balise p est utilisée pour définir un paragraphe de texte.
    • La balise a est utilisée pour créer un lien vers une
      autre page Web ou une autre ressource.
    • Les balises h1 à h6 sont utilisées pour définir les
      en-têtes de section, le h1 étant le plus important et le h6 le moins important.
    • La balise li est utilisée pour définir un élément de liste,
      que ce soit une liste ordonnée ol ou une liste non ordonnée ul .
    • La balise ol est utilisée pour créer une liste ordonnée,
      c'est-à-dire une liste numérotée.
  5. Comment est structurée une page HTML ?

    6. Une page HTML est structurée en utilisant plusieurs balises qui
    permettent d'organiser le contenu de la page. Ce sont généralement
    les balises utilisées dans la question précédente

  6. Qui a créé le web et le langage HTML ?

    7. Le World Wide Web a été créé par Sir Tim Berners-Lee en 1989, alors
    qu'il travaillait pour le CERN en Suisse. Il a inventé le langage
    HTML et le protocole HTTP pour créer et transmettre des pages Web.
    Le premier site Web a été créé en 1991 par Berners-Lee, qui a mis en
    ligne une page d'informations sur le projet World Wide Web.

  7. Qu'est-ce que le w3c ? Ajouter un lien permettant d'accéder à son site web sous votre réponse.

    8. Le W3C développe des normes pour le Web pour assurer l'interopérabilité et
    l'accessibilité. Il est composé de membres de l'industrie, du monde
    universitaire et des gouvernements, et son site web est celui ci.

  8. A quoi sert le site www.w3schools.com ? Ajouter un lien permettant d'y accéder sous votre réponse.

    9. Le site www.w3schools.com est un site web éducatif en ligne qui propose des
    tutoriels, des exemples de code, des quiz et des références pour différentes
    technologies Web, notamment HTML, CSS, JavaScript, SQL, Python et bien d'autres encore.
    Il est souvent utilisé par les développeurs Web débutants et expérimentés pour
    apprendre de nouvelles compétences et pour résoudre des problèmes techniques.
    on peut y accéder via ce lien.

II. Statique ou dynamique

  1. Lire : https://fr.wikipedia.org/wiki/Page_web_statique
  2. Qu'est-ce q'un site web statique ?

    3. Un site web statique est un site web composé de pages web en HTML dont le contenu
    ne change pas fréquemment. Il est créé à l'aide d'un éditeur de texte et peut être
    hébergé sur un serveur Web standard. Les sites web statiques sont simples et ne
    nécessitent pas de connaissances en programmation ou en bases de données, mais
    ils ont des limitations en termes de personnalisation et de fonctionnalités
    interactives.

  3. A quoi l'oppose-t-on ? Détailler votre réponse.

    4. Un site web dynamique est généré à partir d'une base de données ou d'autres sources
    de données et est constamment mis à jour en temps réel en fonction de l'interaction
    de l'utilisateur avec le site. Les sites web dynamiques sont souvent construits en
    utilisant des langages de programmation côté serveur tels que PHP, Python,
    Ruby ou Java, et peuvent offrir des fonctionnalités avancées telles que des
    formulaires interactifs, des zones de commentaires, des paniers d'achat et
    des systèmes de gestion de contenu. Cependant, la création d'un site web dynamique
    nécessite une expertise plus avancée en matière de programmation et de bases de données.

  4. Quels sont les trois langages de programmation utilisés dans un site statique ?

    5. les trois langages de programmation utilisés sont le html le css et le javascript.

  5. A quoi servent-t-ils ?

    6. Le HTML fournit la structure et le contenu de base d'une page web, le CSS est utilisé
    pour contrôler la présentation visuelle de la page et le JavaScript permet d'ajouter
    des fonctionnalités interactives.

  6. Où se situent ces fichiers pour ce site web que vous êtes en train de consulter et modifier.

    7. Les fichiers du site we sont situés dans un même dossier séparés en d'autres dossiers
    et ces fichiers communiquent entre eux grâce au balisage script ou link.
